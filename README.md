# YOUR DEATH PREDICTION — LIFE-2-VEC

## Purpose
A cinematic, sci-fi themed life-awareness experience disguised as a mortality prediction tool.
Users input lifestyle metrics (age, sleep, screen time, exercise) and receive a dramatic,
orbital-UI-powered projection of their lifespan. Designed to create emotional impact and
inspire reflection on time, habits, and what truly matters. Not a medical tool.

---

## Tech Stack
- React 18 + Web Audio API
- Pure CSS animations (no Framer Motion dependency needed)
- Canvas-based cosmic nebula particle background
- Orbitron + Rajdhani + Share Tech Mono (Google Fonts)

---
